Academic Projects

IE4100R BEng. Dissertation

On the Boundary between Dedicated and Pooled Queuing Structures in Large Service Systems

Created a C++ program to simulate two types of queuing structures in large service systems. The performances of both structures were then compared based on multiple performance measures to determine the better structure. Factors such as customer abandonment and different routing policies not considered in traditional queuing theory were incorporated in the simulation to provide useful insights for system designers.

IE3100M System Design Project

Emergency Department Junior Doctor Rostering using Constraint Programming

Partnered with a local hospital to determine a more efficient way of rostering junior doctors in the emergency department. The final product delivered was a web application created using Google OR Tools, python and the flask framework for the back-end development. HTML, CSS, Javascript as well as the React.js library was used for the front-end development. Our web application managed to significantly reduce the time required to generate an optimal roster from 2 weeks, manually, to about 2 hours on average.

Passion Projects

Summarizer Application to Enhance Learning and Retention of Information

(In progress)

I took and completed an edX MIT course on User Innovation & Entrepreneurship. The main deliverable of the course was to determine an important problem that I face in my life and work on developing a solution for it. The problem I decided to work on was that I found it difficult to absorb information from a book or a paper by simply reading it once. I believed that it is crucial that we revisit information learnt constantly so that we may retain them in the long run. I created a survey, interviewed several people and found that summaries were an effective way to revisit information learned but took up too much time. I envision a product where users are able to note down important bits of information and after finishing the material, is able to revisit the information with an automatically generated summary. As a prototype, I researched on how to create a more efficient snipping tool for Windows so that I may incorporate it into my final product. This working snipping tool application was created using PyQt5 and Pillow and can be viewed here on my Github page.
